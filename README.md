# Random Quote Generator

## Автор
Шульга Илья

## Описание
Графическое приложение для генерации и просмотра случайных цитат. Программа позволяет получать случайные высказывания из предустановленной базы, фильтровать их по автору и теме, а также вести историю сгенерированных цитат с возможностью сохранения и загрузки данных в формате JSON.

## Возможности
- Генерация случайной цитаты по нажатию кнопки.
- Отображение истории сгенерированных цитат в виде списка.
- Фильтрация цитат по автору и/или теме.
- Сохранение истории в локальный файл формата JSON.
- Автоматическая загрузка истории при запуске приложения.
- Валидация вводимых данных и проверка на уникальность записей в истории.

## Примеры использования
1. Запустите приложение: `python "Random Quote Generator.py"`
2. Нажмите кнопку «Сгенерировать цитату» для получения случайного высказывания.
3. Используйте поля фильтрации для поиска цитат по автору или теме, затем нажмите «Применить фильтр».
4. История сгенерированных уникальных цитат отображается в списке и сохраняется автоматически.

## Технологии
- Python — язык программирования.
- Tkinter — стандартная библиотека для создания графического интерфейса (GUI).
- JSON — формат для сохранения и загрузки данных истории.
- random — модуль для генерации случайных элементов.

## Запуск тестов (по желанию)
Для проверки корректности работы приложения рекомендуется выполнить следующие действия:
- Сгенерируйте 5–10 различных цитат, чтобы убедиться, что история пополняется.
- Используйте фильтр по автору (например, «Шекспир») и убедитесь, что выводятся только релевантные цитаты.
- Закройте и повторно запустите приложение, чтобы проверить корректность загрузки истории из файла quotes.json.
- Попробуйте применить фильтр, по которому нет совпадений, и убедитесь, что программа корректно обрабатывает эту ситуацию (выводит информационное сообщение).
